# Personal Portfolio — Muskan Ahmed

A fully responsive personal portfolio website built with HTML and CSS.

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Home — hero section, logo strip, and services overview |
| `profile.html` | About me, education, experience, and social links |
| `gallery.html` | Photo grid with hover captions |
| `goals.html` | Short-term and long-term career goals |
| `projects.html` | Project cards with tech stacks and GitHub links |
| `skills.html` | Skill progress bars and interests |
| `contact.html` | Contact form and professional details |

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, animations, media queries)
- Font Awesome 6 (icons)

## Design

Dark minimal theme inspired by modern developer portfolios. Features a scrolling logo marquee, gradient typography, glass-effect cards, and smooth hover transitions.

## Run Locally

No build steps needed. Just open `index.html` in your browser.
